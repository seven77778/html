HTML DOM textContent 与 innerHTML的区别 - 一个是文本，一个是html，带格式的
HTML DOM textContent：设置或者返回指定节点的文本内容。 HTML DOM innerHTML：设置或返回表格行的开始和结束标签之间的 HTML。

  • textContent 是 Node 对象的属性；
  • innerHTML 是 Element 对象的属性；
  • innerText 是 HTMLElement 对象的属性；
